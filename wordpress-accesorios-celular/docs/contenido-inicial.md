## Imagenes incluidas

Las imagenes quedaron en:

```text
assets/catalogo/
assets/catalogo/originales/
```

Puedes subirlas a Medios en WordPress y asignarlas a cada producto. La carpeta `assets/catalogo/` tiene las imagenes principales con nombres limpios para productos. La carpeta `assets/catalogo/originales/` conserva todas las fotos de fundas que pasaste.

Tambien hay copias de cuatro imagenes principales dentro del tema para que la portada se vea bien apenas actives el shortcode `[movilshop_hero]`.
